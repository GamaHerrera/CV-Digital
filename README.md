## Instalación y uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/cv-digital.git
   ```
2. Navega al directorio:
   ```bash
   cd cv-digital
   ```
3. Abre `index.html` en tu navegador o ejecuta un servidor local (opcional):
   ```bash
   npm install -g http-server
   http-server .
   ```
